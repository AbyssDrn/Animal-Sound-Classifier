# TROUBLESHOOTING:
# ============================================================================
#
# ImportError: No module named 'tensorflow':
#   → pip install --upgrade tensorflow
#
# CUDA/GPU not detected:
#   → pip install tensorflow-gpu
#   → Ensure NVIDIA drivers and CUDA toolkit are installed
#
# Audio file format not supported:
#   → pip install --upgrade librosa soundfile
#   → Ensure FFmpeg is installed (apt-get install ffmpeg)
#
# Image download fails (403 Forbidden):
#   → requests library with User-Agent headers (already in code)
#   → Check internet connection
#
# Memory errors during training:
#   → Reduce batch_size in code (default: 32)
#   → Reduce MAX_SAMPLES_PER_SPECIES
#   → Use GPU (install tensorflow-gpu)
